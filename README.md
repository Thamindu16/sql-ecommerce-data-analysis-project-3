# SQL-Based E-Commerce Data Analysis Using Python and SQLite

----- Project Overview -----

    This project was completed as part of my Data Analytics Internship at DecodeLabs.
    
    The main objective of this project was to perform SQL-based analysis on an E-Commerce dataset using Python, Pandas, and SQLite. Before starting the analysis, the dataset was cleaned by checking missing values and duplicate records to improve data quality.
    
    The cleaned dataset was then stored in an SQLite database, where different SQL queries were used to extract meaningful insights related to customer transactions, product performance, payment methods, referral sources, and coupon usage.
    
    This project provided practical experience in database management, SQL querying, and data analytics using real-world business data.

----- Technologies Used -----

    * Python
    * Pandas
    * SQLite
    * SQL
    * Google Colab
    * Microsoft Excel

----- Data Cleaning Process -----

    The following preprocessing steps were performed before SQL analysis:
    
    * Dataset inspection
    * Missing value detection
    * Handling missing values
    * Duplicate record checking
    * Dataset validation
    * Exporting the cleaned dataset

----- SQL Analysis Performed -----

    The following SQL operations were carried out during the project:
    
    * SELECT queries
    * WHERE clause filtering
    * ORDER BY sorting
    * GROUP BY analysis
    * COUNT aggregation
    * SUM aggregation
    * AVG aggregation
    * Multiple condition filtering

----- Key Findings -----

    The analysis provided several useful insights from the dataset:
    
    * Identified the most frequently purchased products
    * Analyzed customer payment preferences
    * Evaluated product revenue performance
    * Examined referral source effectiveness
    * Analyzed coupon usage patterns
    * Identified high-value customer transactions

----- How to Run the Project -----

    1. Open the notebook in Google Colab.
    2. Upload the dataset file.
    3. Run all notebook cells in sequence.
    4. Create the SQLite database.
    5. Execute the SQL queries.
    6. Review the generated outputs and analysis results.
